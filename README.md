# KayKey - Менеджер паролей
Школьный Проект по теме: Менеджер паролей

Руководство пользователя KayKey:
--------------------------------------------------------------------------------------------------------------------------------
KayKey – это консольный менеджер паролей, написанный на Python.
Программа хранит пароли в зашифрованном виде и требует мастер-пароль для доступа.
Подходит для безопасного хранения учётных данных от различных сервисов.
--------------------------------------------------------------------------------------------------------------------------------
1. Установка и требования(требования для .py формата)

Необходимые компоненты:
Python версии 3.6 или выше.
Установленные библиотеки:
 - cryptography
 - colorama
 - pyperclip

Установка библиотек:
Откройте терминал (командную строку) и выполните:

pip install cryptography colorama pyperclip
(Это установит все 3 зависимости(библиотеки))

Скачивание программы:
Скачайте с GitHub (https://github.com/ttostyk/KayKeyPasswordManager) Скриптовую версию(для модификации, и пользования на Linux системах)
Либо же скачайте с Google Disk (https://drive.google.com/file/d/1av97ALxRSCqhMWbZhbOmB35Ifvo8BYh_/view?usp=sharing) .exe исполняемый файл(программа, для систем семейства Виндовс(Windows))
--------------------------------------------------------------------------------------------------------------------------------
2. Первый запуск

Запустите программу(Для Linux систем):
python KayKey.py

Запустите программу(Для Windows систем):
Двойной клик по приложению.

Если файлы хранилища (master.key и passwords.enc) отсутствуют, программа предложит придумать мастер-пароль:
Придумайте мастер-пароль(ключ шифра): 
Введите надёжный пароль (ниже объясняется, как он используется). При вводе символы не отображаются – это нормально.

После создания мастер-пароля появится сообщение: "Хранилище создано!".

> ⚠️ Важно: мастер-пароль нельзя восстановить. Если вы его забудете, доступ к паролям будет потерян навсегда.
--------------------------------------------------------------------------------------------------------------------------------
3. Работа с программой

После успешного входа вы увидите главное меню:

1. Показать все
2. Добавить запись
3. Удалить запись
4. Информация
5. Выход


3.1. Просмотр записей (1)

Выводит список всех сервисов с логинами, пароли скрыты:

  1. Google | mylogin@gmail.com | [пароль скрыт]
  2. GitHub | username | [пароль скрыт]

- Для просмотра пароля введите номер записи (например, 1). Будет показан пароль.
- После показа пароля можно:
  - Скопировать пароль в буфер обмена (выберите 1)
  - Вернуться назад (выберите 2)

3.2. Добавление записи (2)

Введите данные:

Сервис: Google
Логин: myemail@gmail.com
Пароль (Enter = сгенерировать): 

- Если оставить поле пароля пустым и нажать Enter, программа сама сгенерирует надёжный пароль (12 символов: буквы, цифры, спецсимволы !@#$%^&*).
- Можно ввести свой пароль вручную.
- После добавления запись сохраняется в зашифрованное хранилище.

3.3. Удаление записи (3)

- Выводит список сервисов с номерами.
- Введите номер записи для удаления.
- Запись будет безвозвратно удалена.

3.4. Информация (4)

Показывает:
- Версию программы (1.82.Console/EXE.Version)
- Ссылку на GitHub (автоматически копируется в буфер обмена)
- Имя создателя
- Примечание: "вы можете сохранить файлы master.key и passwords.enc на любой носитель, чтобы перенести хранилище на другой компьютер. Для восстановления достаточно скопировать эти файлы в папку с программой."

3.5. Выход (5)

Завершает работу программы.

4. Файлы программы и безопасность

Программа создаёт два файла в той же папке, где находится KayKey.py:

| Файл            | Назначение                                                                              |
|-----------------|-----------------------------------------------------------------------------------------|
| master.key      | Содержит мастер-ключ (производную от мастер-пароля). Никому не показывайте этот файл!   |
| passwords.enc   | Зашифрованный JSON-файл со всеми вашими записями (сервис, логин, пароль).               |

Как работает защита:
- Мастер-пароль не хранится в открытом виде. Он преобразуется в ключ с помощью base64 и дополнения до 32 байт.
- Все данные шифруются симметричным алгоритмом Fernet (на базе AES-128 в режиме CBC).
- При каждом запуске программа сверяет введённый мастер-пароль с ключом из master.key.

Рекомендации:
- Не удаляйте файлы master.key и passwords.enc – иначе пароли будут утеряны.
- Регулярно делайте резервные копии этих двух файлов (например, на флешку или в облачное хранилище). Для восстановления просто скопируйте их обратно в папку с программой.
- Используйте сложный мастер-пароль (не менее 8 символов, включая цифры, буквы разного регистра и спецсимволы).
--------------------------------------------------------------------------------------------------------------------------------
5. Возможные ошибки и их решение

| Сообщение                        | Причина и решение                                                                          |
|----------------------------------|--------------------------------------------------------------------------------------------|
| Неверный пароль!                 | Вы ввели неправильный мастер-пароль. Проверьте раскладку клавиатуры и Caps Lock.           |
| Файл повреждён!                  | Файл passwords.enc был изменён или испорчен. Восстановите его из резервной копии.          |
| ImportError: No module named...  | Не установлена нужная библиотека. Установите через pip install ....                        |
| Программа не запускается         | Убедитесь, что файл KayKey.py находится в отдельной папке и у вас есть права на запись.    |
--------------------------------------------------------------------------------------------------------------------------------
6. Дополнительные возможности

Генерация паролей:
Если при добавлении записи нажать Enter без ввода пароля, программа сама создаст надёжный пароль. Его можно сразу скопировать (см. пункт 3.1).

Копирование в буфер обмена:
- При просмотре пароля можно скопировать его в буфер обмена.
- Ссылка на GitHub в разделе «Информация» также копируется автоматически.


================================================================================================================================
Перенос хранилища на другой компьютер:
1. Скопируйте файлы master.key и passwords.enc на внешний носитель.
2. На новом компьютере установите Python и библиотеки.
3. Поместите файлы в ту же папку, что и KayKey.py.
4. Запустите программу и введите мастер-пароль – все данные будут доступны.
================================================================================================================================


7. Заключение
KayKey – простой и безопасный инструмент для хранения паролей. Он не отправляет данные в интернет, не имеет бэкдоров и полностью работает локально. Помните: безопасность ваших паролей зависит от надёжности мастер-пароля и сохранности файлов master.key и passwords.enc.
